# [定时获取bing图片](https://github.com/rguo97/Spiders/blob/master/Little_Sqiders/bing.py)
这个程序主要实现了每天获取bing的图片，然后保存到本地，累计会保存10天，如果超过十天就会删掉时间按最久的那张，然后还实现了一些日志功能，将程序运行的信息保存在本地。但是这个程序不不能实现自动创建图片保存地址和日志存放地址，所以还需要完善。
